
# Internship Projects - Sneha Devnani

This repository contains two small React-based projects completed during the internship.

---

## 📌 1. Weather App (`weather-app/`)

A simple weather app built using React and OpenWeatherMap API. It allows the user to enter a city name and view current temperature and weather description.

### 🔧 How to Run

```bash
cd weather-app
npm install
npm start
```

---

## 📌 2. Todo List (`todo-list/`)

A basic todo-list app to add and delete tasks. It uses `useState` to manage the list dynamically.

### 🔧 How to Run

```bash
cd todo-list
npm install
npm start
```

---

> ✅ Make sure Node.js and npm are installed before running the apps.
